# 평수계산기 앱 기획서 (바이브코딩용)

## 1. 앱 개요

부동산에서 자주 사용하는 면적 단위인 **평(坪)**을 다양한 단위와 상호 변환하는 계산기 앱.
한국 부동산 시장에서 평 단위가 여전히 관행적으로 사용되므로, 빠르고 직관적인 변환 도구를 제공한다.

---

## 2. 핵심 기능 (4개 탭)

### 탭 1: 제곱미터(m²) ↔ 평
- **양방향 변환** 지원
- 왼쪽 패널: 제곱미터(m²) → 평으로 변환
- 오른쪽 패널: 평 → 제곱미터(m²)로 변환
- 변환 공식: `1평 = 3.305785 m²`
- 예시: 34평 입력 → 112.3967 m² 표시

### 탭 2: 헥타르(ha) ↔ 평
- **양방향 변환** 지원
- 왼쪽 패널: 헥타르(ha) → 평으로 변환
- 오른쪽 패널: 평 → 헥타르(ha)로 변환
- 변환 공식: `1ha = 3,024.9 평`

### 탭 3: 에이커(ac) ↔ 평
- **양방향 변환** 지원
- 왼쪽 패널: 에이커(ac) → 평으로 변환
- 오른쪽 패널: 평 → 에이커(ac)로 변환
- 변환 공식: `1ac = 1,224.17 평`

### 탭 4: (넓이 × 높이) → 평
- 넓이(m)와 높이(m)를 각각 입력
- 두 값의 곱을 m² 면적으로 계산 후 평으로 변환
- 예시: 넓이 120m × 높이 60m = 7,200m² = **2,178평**

## 3. UI/UX 설계

### 3-1. 전체 레이아웃 (위→아래)

```
┌─────────────────────────────┐
│         헤더 (앱 타이틀)       │
├───┬───┬───┬───────────────────┤
│탭1│탭2│탭3│     탭4           │  ← 탭 네비게이션
├───┴───┴───┴───────────────────┤
│  ┌──────────┬──────────┐      │
│  │ 왼쪽 패널 │ 오른쪽 패널│      │  ← 변환 결과 영역
│  │ (입력/결과)│(입력/결과) │      │
│  └──────────┴──────────┘      │
├─────────────────────────────┤
│                               │
│        숫자 키패드             │  ← 입력 영역
│     7   8   9                │
│     4   5   6                │
│     1   2   3                │
│     .   0   ←                │
│                               │
├─────────────────────────────┤
│      변환 실행 버튼            │  ← 하단 액션 바
└─────────────────────────────┘
```

### 3-2. 변환 결과 영역 (상단 패널)

| 구성 요소 | 설명 |
|----------|------|
| **왼쪽 패널** | 진한 청록색(teal) 배경, 물결 아이콘(≈), 숫자 표시, 단위 라벨 |
| **오른쪽 패널** | 연두색(light green) 배경, 연필 아이콘(✏️), 숫자 표시, 단위 라벨 |
| **활성 패널** | 현재 입력 중인 패널에 연필 아이콘 표시 |
| **비활성 패널** | 물결 아이콘 표시 (결과가 자동 표시됨) |

- 패널 클릭 시 입력 대상 전환 (양방향 변환)
- 탭4는 넓이/높이 각각 별도 입력

### 3-3. 숫자 키패드

- 3×4 그리드 배열
- 연두색 배경, 흰색 큰 숫자
- 소수점(`.`), 0, 백스페이스(`←`) 포함
- 키 터치 시 햅틱 피드백 (선택)

### 3-4. 하단 변환 버튼

- 남색(navy) 배경, 흰색 텍스트
- 현재 변환 방향 텍스트 동적 표시
  - 예: "평수를 제곱미터(m²)로 변환하기"
  - 예: "(넓이×높이)를 평수로 변환하기 → 2,178평"

## 4. 변환 공식 정리

```
1평 = 3.305785 m²
1평 = 3.305785 × 10⁻⁴ ha
1평 = 0.000816887 ac

역변환:
1 m²  = 0.3025 평
1 ha  = 3,024.9 평
1 ac  = 1,224.17 평

탭4 계산:
평 = (넓이m × 높이m) / 3.305785
```

## 5. 사용자 인터랙션 흐름

### 양방향 변환 (탭 1~3)

```
1. 사용자가 탭 선택 (예: 제곱미터 ↔ 평)
2. 기본 입력 패널: 오른쪽 (평 입력)
3. 숫자 키패드로 값 입력
4. 실시간으로 반대쪽 패널에 변환 결과 표시
   (또는 하단 버튼 클릭 시 변환)
5. 반대쪽 패널 터치 → 입력 대상 전환
6. CLEAR 버튼 → 초기화
```

### 넓이×높이 변환 (탭 4)

```
1. 탭4 선택
2. 넓이(m) 패널 선택 후 숫자 입력
3. 높이(m) 패널 선택 후 숫자 입력
4. 하단에 자동으로 평수 계산 결과 표시
5. CLEAR 버튼 → 초기화
```

## 6. 추가 기능 (선택)

- **CLEAR 버튼**: 우측 상단, 모든 입력값 초기화
- **복사 기능**: 결과값 길게 누르면 클립보드 복사
- **천 단위 콤마**: 결과값에 자동 천 단위 구분 (예: 2,178평)
- **소수점 자릿수 설정**: 소수점 이하 표시 자릿수 선택 가능
- **다크모드 지원**: 시스템 설정 연동

## 11. 핵심 변환 로직 (JavaScript)

```javascript
const PYEONG_TO_SQM = 3.305785;
const PYEONG_TO_HA = 0.0003305785;
const PYEONG_TO_AC = 0.000816887;

// 평 → 제곱미터
const pyeongToSqm = (pyeong) => pyeong * PYEONG_TO_SQM;

// 제곱미터 → 평
const sqmToPyeong = (sqm) => sqm / PYEONG_TO_SQM;

// 평 → 헥타르
const pyeongToHa = (pyeong) => pyeong * PYEONG_TO_HA;

// 헥타르 → 평
const haToPyeong = (ha) => ha / PYEONG_TO_HA;

// 평 → 에이커
const pyeongToAc = (pyeong) => pyeong * PYEONG_TO_AC;

// 에이커 → 평
const acToPyeong = (ac) => ac / PYEONG_TO_AC;

// 넓이 × 높이 → 평
const areaToPyeong = (width, height) =>
  (width * height) / PYEONG_TO_SQM;
```
