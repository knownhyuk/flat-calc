## 전체 설명 (4000자 이내)
평수계산기는 부동산 면적 단위를 빠르고 정확하게 변환해주는 앱입니다.

아파트 분양, 부동산 매매, 토지 거래 시 꼭 필요한 면적 단위 변환을 한 번의 터치로 해결하세요.

주요 기능:

[제곱미터 ↔ 평 변환]
m²를 평으로, 평을 m²로 실시간 양방향 변환

[헥타르 ↔ 평 변환]
ha를 평으로, 평을 ha로 간편 변환

[에이커 ↔ 평 변환]
ac를 평으로, 평을 ac로 변환

[면적 계산 (넓이×높이)]
가로, 세로 길이를 입력하면 면적(m²)과 평수를 동시에 계산

[국민평형 아파트]
59타입, 84타입 등 아파트 전용면적별 평수를 바로 확인

특징:
- 양방향 실시간 변환 (입력 즉시 결과 표시)
- 직관적인 키패드로 빠른 입력
- 설정 기능: 기본 입력 단위 및 시작 탭 선택
- 깔끔하고 심플한 디자인
- 인터넷 연결 없이 사용 가능 (오프라인 지원)

이런 분들에게 추천합니다:
- 아파트 분양/매매 시 평수가 궁금한 분
- 부동산 중개업 종사자
- 토지, 농지 면적을 계산해야 하는 분
- 건축, 인테리어 관련 업무를 하시는 분

## 비공개 테스트 진행 순서
1. Google Play Console 접속 (https://play.google.com/console)
2. 앱 만들기 → 앱 이름, 언어, 카테고리 입력
3. 대시보드 → 앱 설정 작업 완료 (개인정보처리방침, 콘텐츠 등급 등)
4. 테스트 → 비공개 테스트 → 새 버전 만들기
5. app-release.aab 업로드
6. 테스터 이메일 목록 추가
7. 검토 요청
